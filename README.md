## Launch

 How to Launch?

 Write your assembly code in **src.txt** in the same project directory and:

 for SIC Shell, for the first time:
 ```bash
 $ g++ -o run run.cpp
 $ ./run
 ```
 To run the shell in general:
 ```bash
 $ ./run
 ```
